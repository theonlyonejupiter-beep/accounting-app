# 💰 个人记账本 - 项目规划

## 项目概述
一个简洁易用的个人记账本Web应用，支持微信和支付宝账单导入，专注于记账准确性和实用性。

## 技术栈
- **前端**: HTML5 + CSS3 + JavaScript (原生，无框架)
- **存储**: LocalStorage (本地存储，无需后端)
- **图标**: Font Awesome
- **编码**: 支持GBK编码（微信/支付宝账单）

## 💰 金额计算说明

### 问题
JavaScript的浮点数计算会有精度问题：
```javascript
0.1 + 0.2 = 0.30000000000000004
```

### 解决方案
所有金额计算转换为整数（分）进行运算：
```javascript
// 加法
add(a, b) {
    const centsA = Math.round(parseFloat(a) * 100);
    const centsB = Math.round(parseFloat(b) * 100);
    return (centsA + centsB) / 100;
}

// 结果
add(0.1, 0.2) = 0.3  // 正确！
```

## 📥 账单格式说明

### 支持的文件格式
- **Excel格式**（.xlsx）- 微信和支付宝导出的默认格式
- **CSV格式**（.csv）- 备用格式

### 微信账单格式
Excel表格，包含以下列：
```
交易时间 | 交易类型 | 交易对方 | 商品 | 收/支 | 金额(元) | 支付方式 | 当前状态 | 交易单号 | 商户单号 | 备注
```

### 支付宝账单格式
Excel表格，包含以下列：
```
交易号 | 商家订单号 | 交易创建时间 | 付款时间 | 最近修改时间 | 交易来源地 | 类型 | 交易对方 | 商品名称 | 金额（元） | 收/支 | 交易状态 | 服务费（元） | 成功退款（元） | 备注 | 资金状态
```

## 🎯 设计原则

1. **准确性优先**：金额计算精确到分，确保账目对得上
2. **实用为主**：支持国内最常用的微信和支付宝
3. **简洁易用**：界面清爽，操作直观
4. **数据安全**：本地存储，隐私保护
5. **响应式**：适配各种屏幕尺寸
