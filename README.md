# 💰 我的记账本

一个简洁易用的个人记账本Web应用，支持微信和支付宝账单导入，支持PWA安装到手机桌面。

## ✨ 核心功能

### 1. 精准记账 ✅
- **金额精确到分**：使用整数运算，避免浮点数精度问题
- **多账户支持**：微信、支付宝、现金分开统计
- **收支总览**：一目了然查看各账户收支情况

### 2. 账单导入 ✅
- **微信账单导入**：支持微信导出的Excel(.xlsx)格式
- **支付宝账单导入**：支持支付宝导出的Excel(.xlsx)格式
- **CSV格式支持**：也支持CSV格式
- **智能去重**：避免重复导入
- **自动分类**：根据交易描述智能判断分类

### 3. 日期查询 ✅
- **按日期筛选**：选择任意日期范围查看
- **快捷选择**：今天、本周、本月、本年
- **按账户筛选**：单独查看微信/支付宝/现金
- **日期分组**：按日期分组显示，清晰明了

### 4. 数据统计与图表 ✅
- **支出分类饼图**：直观查看各分类支出占比
- **收入分类饼图**：了解收入来源分布
- **每日收支趋势**：查看收支变化趋势
- **分类占比**：每个分类的百分比显示

### 5. 预算管理 ✅
- **月度预算**：设置每月支出预算上限
- **进度条显示**：直观查看预算使用情况
- **超支提醒**：80%、90%、100%时自动提醒
- **快捷设置**：一键设置常用预算金额

### 6. 数据导出 ✅
- **CSV导出**：可在Excel中打开编辑
- **JSON导出**：用于数据备份和恢复
- **中文支持**：完美支持中文字符

### 7. 深色模式 ✅
- **一键切换**：点击右上角月亮/太阳图标
- **自动跟随**：首次使用自动跟随系统主题
- **记忆功能**：保存您的主题偏好
- **完整适配**：所有界面都支持深色模式

### 8. 搜索功能 ✅
- **实时搜索**：输入即搜索，无需回车
- **多字段搜索**：支持搜索交易对方、备注、分类
- **防抖优化**：输入时自动防抖，避免频繁搜索
- **一键清除**：点击X按钮清除搜索内容

### 9. 语音记账 ✅
- **语音输入**：点击麦克风按钮，说出"午饭35元"
- **智能识别**：自动解析金额和描述
- **自动分类**：根据描述智能判断分类
- **中文支持**：完美支持中文语音

### 10. 常用模板 ✅
- **保存模板**：将常用记录保存为模板
- **一键记账**：点击模板快速填充表单
- **模板管理**：支持删除不需要的模板
- **最多10个**：自动限制模板数量

### 11. 记录管理 ✅
- **新增记录**：快速记一笔
- **编辑记录**：点击记录即可修改
- **删除记录**：支持删除不需要的记录
- **分类选择**：12种支出分类 + 7种收入分类

## 🚀 快速开始

### 方法一：直接打开
双击 `index.html` 文件即可使用

### 方法二：使用本地服务器
```bash
cd 记账本
python -m http.server 8080
# 打开 http://localhost:8080
```

### 方法三：安装到手机（PWA）📱

**安卓手机安装步骤：**

1. **启动本地服务器**
   ```bash
   cd 记账本
   python -m http.server 8080
   ```

2. **获取电脑IP地址**
   - Windows: 打开命令提示符，输入 `ipconfig`，找到 IPv4 地址
   - Mac/Linux: 打开终端，输入 `ifconfig`，找到 inet 地址

3. **手机连接同一WiFi**
   - 确保手机和电脑连接到同一个WiFi网络

4. **手机浏览器访问**
   - 打开手机Chrome浏览器
   - 访问 `http://电脑IP:8080`（例如：http://192.168.1.100:8080）

5. **添加到主屏幕**
   - 点击Chrome菜单（右上角三个点）
   - 选择「添加到主屏幕」或「安装应用」
   - 输入应用名称（如：记账本）
   - 点击「添加」

6. **完成安装**
   - 返回手机主屏幕
   - 找到「记账本」图标
   - 点击即可像原生App一样使用

**PWA特性：**
- ✅ 全屏运行，无地址栏
- ✅ 支持离线使用
- ✅ 启动速度快
- ✅ 数据本地存储，隐私安全

## 📖 使用指南

### 手动记账
1. 点击右上角「记一笔」
2. 选择「支出」或「收入」
3. 输入金额
4. 选择账户（微信/支付宝/现金）
5. 选择分类
6. 选择日期
7. 添加备注（可选）
8. 点击「保存」

### 语音记账
1. 点击右上角「记一笔」
2. 点击「点击说话」按钮
3. 说出记录，如"午饭35元"或"打车花了22块"
4. 系统自动识别并填充表单
5. 确认无误后点击「保存」

**支持的语音格式**：
- "午饭35元"
- "打车花了22块"
- "买咖啡15.5"
- "收入工资8000元"

### 使用常用模板
1. 点击右上角「记一笔」
2. 在「常用记录」区域点击已有模板
3. 表单自动填充
4. 确认无误后点击「保存」

**保存模板**：
1. 输入金额、选择分类、填写备注
2. 点击「存为模板」按钮
3. 下次可直接使用

### 导入微信账单
1. 打开微信 → 「我」→「服务」→「钱包」
2. 点击右上角「账单」
3. 点击右上角「...」→「下载账单」
4. 选择「用于个人对账」
5. 设置时间范围，输入邮箱
6. 在邮箱中下载Excel文件（.xlsx）
7. 在记账本中点击「导入账单」→ 选择文件

### 导入支付宝账单
1. 打开支付宝APP → 「我的」→「账单」
2. 点击右上角「...」→「开具交易流水证明」
3. 或访问支付宝网页版 → 交易记录
4. 筛选时间范围，点击「下载账单」
5. 选择Excel格式下载
6. 在记账本中点击「导入账单」→ 选择文件

### 查询特定日期
1. 在顶部筛选区域选择日期范围
2. 或使用快捷按钮（今天/本周/本月/本年）
3. 可选择特定账户筛选

## 💾 数据说明

### 存储位置
- 所有数据存储在浏览器的 LocalStorage
- 数据不会上传到服务器
- 清除浏览器数据会删除记录

### 金额计算
- 所有金额计算使用整数运算（转换为分）
- 避免JavaScript浮点数精度问题
- 确保收支数字准确对得上
